External Tools
 

Here is a list of *free* third party tools that can be used in conjunction with QuickFIX. These utilities are in no way affiliated with QuickFIX and are supported by their respective companies or organizations. All tools

Product Company Description
Ethereal Ethereal A very powerful and full featured network protocol analyzer which support hundred of protocols, including FIX. Use this to sniff out all FIX traffic regardless of what FIX engine it originates from.
FIXionary Javelin Technologies Interactively browse the FIX protocol specification through this java application. Much nicer then browsing through the massive word documents or XML dictionaries. Can be downloaded or used online.
Stunnel Stunnel.org Allows you to encrypt arbitrary TCP connections inside SSL. Use this to encrypt your FIX sessions without need to modify you're application code.


QuickFIX is a full-featured open source FIX engine, currently compatible with the FIX 4.0-4.4 spec. It runs on Windows, Linux, Solaris, FreeBSD and Mac OS X. API's are available for C++, Java, and .NET
